<h1>Remote Desktop into your virtual machine in Azure</h1>
This project explains how to connect to a virtual machine (VM) in Microsoft Azure using Remote
Desktop using a Windows PC. Assuming you have already created your virtual machine in Azure.
<br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines)
- Remote Desktop
- Windows PC

<h2>Operating Systems Used </h2>

- Windows 11
  
<h2>Login to Azure Portal</h2>
<p>
Go to https://portal.azure.com and sign in with your account. Your home page should look something like this when you login to Azure portal. 

</p>

<h2>Find your Virtual Machine Created</h2>
<p>
Find the virtual machine you previously created. You can click on the 3 lines on the left side of the homepage and click on Virtual Machines or you can also just type virtual machines in the search toolbar. Make sure the virtual machine you want to remot desktop into has a status of running and not stopped in azure.
</p>
